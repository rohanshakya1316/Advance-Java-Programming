# ☕ Advanced Java Programming Lab

A comprehensive collection of **Advanced Java Programming** practical implementations covering:

- Desktop GUI Applications using Java AWT/Swing
- Remote Method Invocation (RMI)
- Java Servlets
- JavaServer Pages (JSP)
- JDBC Database Connectivity

This repository contains academic lab exercises, practical implementations, and mini-projects focused on core enterprise Java development.

## ⚙ Requirements

To run these programs, install:

- Java JDK 8+
- IDE (Recommended):
  - IntelliJ IDEA
  - Eclipse
  - NetBeans
- Apache Tomcat (for Servlet/JSP)
- MySQL (for JDBC examples)

---

## 🚀 How to Run

### Clone Repository

```bash
git clone <your-repository-url>
```

### Compile Java Files

```bash
javac FileName.java
```

### Run Java Program

```bash
java FileName
```

### For RMI Programs

Start RMI registry:

```bash
rmiregistry
```

Run server:

```bash
java Server
```

Run client:

```bash
java Client
```

### For Servlet/JSP

1. Import `JavaServletLab` into Eclipse/IntelliJ
2. Configure Apache Tomcat
3. Run on server
